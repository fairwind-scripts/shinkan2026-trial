### I. コードを書き換えてみよう

ここでは、コードを書き換えてページのどの部分が変わるかを観察し、html・css に関する理解を深めましょう！
例えば、

- `<h2>, <p>, <span>` タグで挟まれている文字を書き換えてみる
- `<h2>, <p>, <span>` タグに class を付与してみる
  - class の例は以下の通りです
    - `font-bold`：太字になります
    - `underline`：下線を引きます
      - 派生形として、`overline, line-through` などもあります
      - class の手前に `hover:` を付けることで、カーソルが合っているときだけ class が適用されます（`hover:underline` など）
    - `text-xl, text-2xl, text-3xl ...`：文字の大きさを変えます
    - `text-white, text-primary`：文字の色を変えます
